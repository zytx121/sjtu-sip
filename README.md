# 上交双创平台使用指北

双创联盟由上海交通大学（或C9高校）发起的纯公益联盟，致力于为全国大学生创新创业搭线上交流平台，促进全国范围大学生之间协同创新，通过项目大数据及企业专家指导促进学生累进创新，联盟提供专业的设施及软硬件条件，实现大学生双创水平上台阶。
双创联盟秘书处设在上海交通大学学生创新中心。

目前已接入34所双一流高校，69所双一流学科高校，覆盖全国200万大学生。

目前已接入99家企业，提供企业需求300项。
